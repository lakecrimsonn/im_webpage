# 인피니트몽키즈 자사 웹페이지 테스트

- 회사 웹페이지는 애니메이션을 이용하여 다이나믹하게 구성하고 싶어서 플러터를 사용하였습니다.
- was, db까지 연결된 프로젝트는 아니고 단순히 프론트만 있는 데모 타입입니다.
- model에는 웹페이지에 표현될 내용들을 임의적으로 넣어놨습니다. 추후 api를 받아오고 비지니스 로직을 처리할 예정입니다.
- MVVM 패턴을 사용했고, 상태관리는 getx를 이용하였습니다.
